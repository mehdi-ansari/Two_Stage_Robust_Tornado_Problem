To set up the parameter open userInputParameter.txt files:
input_data_name, choose: Joplin / Sample1 / Sample2 / Sample3 / Sample4 / Sample5
subproblem_formulation, choose: Original / Decomposed
budget: ANY VALUE
length: ANY VALUE
width: ANY VALUE
number_of_clusters, choose: None / ANY INTEGER

The results will be stored in the Results directory.
